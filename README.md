## GraphQL 簡單實作範例

- 前情準備
    1. ```python3 -m venv demo_graph```
    2. ```source ./demo_graph/bin/activate```
    3. ```pip strawberry-graphql[fastapi]```

- 需要注意的地方
    1. db 連結帳密要換喔
    2. 資料的話....我給你啦～那就生一個data.json，要你自己存進db喔

- 啟動
    ```strawberry server app```
